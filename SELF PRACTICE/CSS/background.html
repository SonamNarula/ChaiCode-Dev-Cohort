CSS Background Properties

1. Background-Size: Cover

Scales content to cover entire div (cropping if needed).

2. Background-Size: Contain

Scales content to be fully visible (may leave empty space).

3. Background-Repeat: Repeat

Repeats image to fill the div.

4. Background-Position: Center

Centers the image within the container.

5. Linear Gradient

A gradient background instead of an image.

6. Background-Attachment: Fixed

The background image stays still while you scroll.
