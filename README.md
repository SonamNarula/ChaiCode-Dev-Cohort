If HTML is the structure, JavaScript is the heartbeat. This phase represents a rigorous deep dive into the invisible logic that drives the machine.
- **Memory & Syntax:** Mastered lexical environments, hoisting, and strict `var/let/const` execution scoping. ([Terminal Engine](./JAVASCRIPT/app.js) / [Syntax Basics](./JAVASCRIPT/script.js) / [Variable Scoping](./JAVASCRIPT/brain.js))

### Phase II: The Abyssal Era (Structural Layouts)
> *"Meet me at midnight..."*

Mastering CSS layouts is complex and infinitely powerful. This is the era of diving into the deep end without a CSS library, relying purely on raw spatial geometry.
- **1D Flexbox Protocol:** Controlled precise alignment along a single axis managing `display: flex`, spatial distribution, and cross-axis positioning. ([Flexbox Laboratory](./SELF%20PRACTICE/CSS/flexbox.html))
- **2D Grid Architecture:** Handled complex multi-axis layouts by defining explicit tracks, responsive fractions (`fr`), and named template areas. ([Grid System](./SELF%20PRACTICE/CSS/grid.html))
