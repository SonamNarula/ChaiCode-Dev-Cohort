JavaScript: Console-First Walkthrough

Open DevTools ➜ Console to watch each chapter log insights and examples.

  1. 01_variables.js — var / let / const, TDZ, hoisting.
  2. 02_data_types.js — primitives vs references, typeof quirks, truthy/falsy.
  3. 03_operators.js — arithmetic, comparison, logical, nullish, optional chaining.
  4. 04_control_flow.js — if/else, guard clauses, switch.
  5. 05_loops.js — for, while, do-while, for-of, forEach, loop control.
  6. 06_functions.js — declarations, expressions, arrows, rest/default, closures.
  7. 07_arrays.js — mutation vs copy, destructuring, map/filter/reduce.
  8. 08_objects.js — dot vs bracket, methods, destructuring, utilities.

Each file is wrapped in an IIFE so they don't leak globals; read along in the console.
