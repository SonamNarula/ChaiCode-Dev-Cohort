In-line Formatting (Semantic vs Stylistic)

There is a massive difference between <b> and <strong>.

  • This is bold via <strong>: It indicates importance or urgency.
  • This is bold via <b>: It's just for aesthetic weight (use sparingly).
  • This is emphasized via <em>: Indicates stress or tone.
  • This is italicized via <i>: Usually for technical terms or thoughts.
