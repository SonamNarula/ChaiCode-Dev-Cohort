Module 01: Document Architecture & Meta-Data

"Code that doesn't explain itself is just static. Code that does is an asset."

1. The <!DOCTYPE html>

This is not a tag; it's a declaration. It tells the browser to use Standards Mode instead of Quirks Mode. Modern web development demands strict adherence to the latest HTML specifications.

2. The <html lang="en"> Root

Always specify the language. This allows Screen Readers (Accessibility) and Search Engines (SEO) to process your text with the correct algorithmic context.

3. The <head> Section (The Brain)

This is where machine-level communication happens. It contains resources that aren't visible to users but are vital for performance, security, and searchability.

  • Charset: Defines character mapping.
  • Viewport: Defines scaling on mobile devices.
  • Meta Description: The snippet users see on Google results.
