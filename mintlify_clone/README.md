# Mintlify Documentation Website Clone

A desktop-first recreation of the [Mintlify](https://mintlify.com) documentation website, focusing on structural accuracy, typography, and visual hierarchy using pure HTML and CSS.

This project was built as part of the ChaiCode Dev Cohort 8-hour challenge.
